Buzzsprout Podcast API

This Buzzsprout episode handler takes podcast episodes hosted on a podcast hosting service and places them on the show's website. It takes JSON data from the API of a podcast hosting service and uses it to dynamically generate web pages for each individual podcast episode. This brings listeners away from the podcast hosting site and into the podcaster’s own domain, allowing them to create spaces for advertisers, links to their Patreon, and establish/reinforce brand standards for their product.
